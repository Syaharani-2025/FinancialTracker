# 💸 Financial Tracker — by Dinda

> "Stop wondering where your money went. NOW YOU'LL KNOW." 😤

---

## 🤔 Why I Built This
Because I kept asking myself "where did my money go??" 
every end of the month and the answer was always 
"idk bestie" 💀

So I built this. Problem solved. You're welcome.

---

## ✨ Features
- 💰 Track your **income** (salary, freelance, from parents lol)
- 🛍️ Track your **expenses** (food, shopping, random stuff)
- 🏦 Track your **savings** (emergency fund, travel, etc)
- 📊 **Recap** — see everything + your balance
- 💾 **Auto saves** your data so you never lose it
- 💡 **Random quote** every time you exit bc you need motivation

---

## 🛠️ Built With
- Java ☕
- Maven 📦
- Gson (by Google) 🔧
- NetBeans IDE 💻
- A lot of patience 😭

---

## 🚀 How to Run
1. Clone this repo
2. Open in NetBeans
3. Run `FinancialTracker.java`
4. Start tracking your broke era 💀
